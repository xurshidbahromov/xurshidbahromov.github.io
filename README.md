## ğŸƒ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/xurshidbahromov/xurshidbahromov.github.io.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd xurshidbahromov-personal-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
